# Using Tensorboards via the CLI

```bash
Usage: gradient tensorboards [OPTIONS] COMMAND [ARGS]...

  Manage tensorboards

Options:
  --help  Show this message and exit.

Commands:
  add-experiments     Update tensorboard experiments
  create              Create new tensorboard
  delete              Delete tensorboard
  details             Show details of a tensorboard
  list                Show list of tensorboards
  remove-experiments  Update tensorboard experiments
```
